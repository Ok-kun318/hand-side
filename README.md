## このプロジェクトを読み込みます

MakeCode でこのリポジトリを読み込みます。

* [https://makecode.microbit.org/](https://makecode.microbit.org/) を開く
* **読み込む** をクリックし、 **URLから読み込む...** をクリックしてください
* **https://github.com/ok-kun318/hand-side** を貼り付けてインポートをクリックしてください
